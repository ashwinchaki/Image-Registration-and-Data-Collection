Download this project as a .zip file Download this project as a tar.gz file

Image Registration and Data Collection

This is a collection of scripts to run ImageJ turboreg on a stack of .png files of neurons.

Required Software:

MATLAB

ImageJ

MIJ (MATLAB-ImageJ Plugin) <-- This will need to be initialized using the javaaddpath command, as detailed on their site. It is already initialized in the script, so this only applies to those attempting to use it without the script.

Export_Fig (MATLAB Script) - add this to the MATLAB path or put it in the default MATLAB folder in \Users\Documents\MATLAB
